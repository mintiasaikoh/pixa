## 🚀 使用方法

1. **サーバーを起動**
   ```bash
   cd /Users/mymac/pixa
   ./start_server.sh
   ```

2. **ブラウザでアクセス**
   - http://localhost:5001 が自動的に開きます

3. **新しいUIを体験**
   - 左パネルですべての設定が完結！
   - アコーディオンをクリックして詳細設定を開閉
   - 品質プリセットをワンクリックで選択
